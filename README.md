облачные технологии позволяют хранить файлы и работать с ними через интернет. они дают доступ к документам с любого устройства. с помощью облачных сервисов можно совместно редактировать файлы и обмениваться информацией. также облака помогают не потерять данные, потому что они хранятся на удалённых серверах.
